# StarTime

A library for managing temporal systems providing comprehensive time tracking and conversion utilities. This project was commissioned by the Blue Institute of Measurements via the Blue Shipyard corporation and the Grand Blue Academy Scientific Excellency Department, under authority of the High Council and House of Blue.

## Purpose

This library works to simplify the handling of temporary systems by handling tracking and conversion of time between Galactic Standard Time (GST) and Imperial Time. Additionally, there is support for adding custom times to account for local planetary times.
